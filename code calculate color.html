การแปลผล (Interpretation)

Color Strength:

  • 95–105% = ผ่าน (Pass)
  • 90–95% = อ่อนเล็กน้อย
  • <90% = สีอ่อน
  • >110% = สีเข้มเกิน

ΔE Interpretation:

  • ≤ 0.8 = Excellent
  • 0.8–1.0 = Pass
  • 1.0–1.5 = Acceptable
  • >1.5 = Fail
  • >3.0 = Clearly visible

สูตรที่ใช้คำนวณ (Formulas)

1. ค่าความเข้มสี (K/S)

K/S = (1 - R)² / (2R)

2. Color Strength (%)

(K/S_after ÷ K/S_before) × 100

3. Color Loss (%)

(1 - (K/S_after ÷ K/S_before)) × 100

4. ΔE (Color Difference)

√[(ΔL)² + (Δa)² + (Δb)²]
